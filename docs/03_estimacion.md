# Estimación del Proyecto

## Suposiciones de la estimación
- El proyecto corresponde a un sistema de complejidad baja a media.
- El desarrollo será realizado por una sola persona.
- Los requisitos iniciales se mantienen estables durante el desarrollo.
- No se consideran integraciones externas ni funcionalidades avanzadas.

## Descomposición funcional y esfuerzo estimado

| Módulo                      | Esfuerzo estimado (horas) |
|----------------------------|--------------------------- |
| Autenticación y roles      | 8                          |
| Gestión de usuarios        | 10                         |
| Gestión de alumnos         | 10                         |
| Gestión de materias        | 10                         |
| Gestión de calificaciones  | 12                         |
| Documentación y pruebas    | 10                         |
| **Total estimado**         | **60 horas**               |

## Nivel de incertidumbre
La estimación presenta un nivel de incertidumbre medio (±20%), debido a posibles ajustes en los requisitos y al nivel de experiencia del desarrollador.
