# Estimación del Proyecto

## Suposiciones de la estimación
- El proyecto es de tamaño pequeño.
- El desarrollo lo realiza una sola persona.
- No se incluyen integraciones externas.
- La estimación es preliminar.

## Descomposición funcional y esfuerzo estimado

| Módulo                       | Horas estimadas |
|------------------------------|-----------------|
| Gestión de usuarios          | 12              |
| Gestión de alumnos           | 10              |
| Gestión de materias          | 10              |
| Registro de calificaciones   | 12              |
| Autenticación y roles        | 8               |
| Documentación y pruebas      | 8               |
| **Total estimado**           | **60 horas**    |

## Nivel de incertidumbre
La estimación tiene un nivel de incertidumbre medio (±20%), debido a posibles ajustes en los requisitos y al nivel de experiencia del desarrollador.
