# Requisitos Funcionales

## RF-01
El sistema deberá permitir registrar alumnos capturando nombre completo, matrícula, carrera y correo institucional.

## RF-02
El sistema deberá permitir editar la información de un alumno previamente registrado.

## RF-03
El sistema deberá permitir registrar materias indicando clave, nombre y créditos.

## RF-04
El sistema deberá permitir asignar materias a alumnos.

## RF-05
El sistema deberá permitir capturar calificaciones numéricas en un rango de 0 a 100.

## RF-06
El sistema deberá permitir consultar el historial académico de un alumno.

## RF-07
El sistema deberá generar reportes de calificaciones por alumno en formato PDF.

## RF-08
El sistema deberá permitir autenticar usuarios mediante usuario y contraseña.
