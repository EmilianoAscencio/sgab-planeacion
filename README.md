# Sistema de Gestión Académica Básico (SGAB)

## Alumno
- Emiliano Elizarras Ascencio

## Descripción
Este repositorio contiene la planeación inicial del proyecto **Sistema de Gestión Académica Básico (SGAB)**, un sistema web orientado a la administración de alumnos, materias, calificaciones y usuarios.

## Objetivo del repositorio
Documentar el análisis previo al desarrollo del sistema aplicando principios de Ingeniería de Software, considerando calidad, estimación, riesgos y alcance, utilizando Git y Markdown como herramientas profesionales.
